# MingDeng 使用指南

## 📋 核心功能详解

### 1. 批量生成学习计划

**适用场景**: 当你想学习一个复杂的技能或知识领域时。

**使用步骤**:

1. **准备学习内容**
   - 详细列出你想学习的所有主题
   - 可以包含具体的技术栈、工具、概念等

   示例:
   ```
   我想学习机器学习，需要掌握：
   - Python 基础（数据结构、函数、面向对象）
   - NumPy 和 Pandas（数据处理）
   - Matplotlib（数据可视化）
   - Scikit-learn（经典算法）
   - 深度学习基础（神经网络、反向传播）
   - TensorFlow 或 PyTorch
   ```

2. **选择学习模式**
   - **🔄 交叉学习**: 不同主题交叉安排，适合长期学习，避免疲劳
   - **🎯 集中攻坚**: 同一主题集中学习，适合短期冲刺，深入掌握

3. **查看和调整计划**
   - AI 生成计划后，你可以预览所有任务
   - 检查任务难度分布
   - 查看每日时长估算

4. **保存计划**
   - 确认无误后点击「确认并保存计划」
   - 计划会自动添加到你的任务列表

**最佳实践**:
- 输入越详细，生成的计划越精准
- 首次使用建议选择「交叉学习」模式
- 生成后可以手动调整任务顺序和日期

---

### 2. 每日任务管理

**适用场景**: 每天查看和完成学习任务。

**使用步骤**:

1. **查看今日任务**
   - 打开「🏠 首页」
   - 默认显示「今日任务」视图
   - 每个任务显示：
     - 难度标记（🟢 简单、🟡 中等、🔴 困难）
     - 预计时长
     - 相关标签
     - 推荐资源（如果有）

2. **完成任务**
   - 完成后点击「✅ 完成」
   - 系统会记录完成时间
   - 完成率会自动更新

3. **跳过任务**
   - 如果今天无法完成，点击「⏭️ 跳过」
   - 任务会自动移到明天
   - 不影响完成率统计

4. **查看本周任务**
   - 切换到「本周任务」视图
   - 按日期查看本周所有任务
   - 提前规划学习安排

**最佳实践**:
- 每天早上打开 MingDeng 查看任务
- 优先完成困难任务（早上精力充沛）
- 合理利用跳过功能，不要有心理负担

---

### 3. 碎片资源管理

**适用场景**: 在网上看到好的学习资源，想保存起来。

**使用步骤**:

1. **保存资源**
   - 点击「📚 资源库」→「💾 保存资源」
   - 粘贴资源链接或输入文本描述
   - 添加简短描述（可选）
   - 勾选「自动关联到相关任务」

2. **AI 自动关联**
   - AI 会分析资源内容
   - 自动匹配到相关学习任务
   - 在执行任务时，会推荐给你

3. **手动管理关联**
   - 查看资源详情
   - 点击「🔗 关联任务」手动关联
   - 或点击「🔓 取消关联」解除关联

4. **更新资源状态**
   - 未读 → 阅读中 → 已完成
   - 方便追踪学习进度

**资源类型自动识别**:
- 🎥 视频: YouTube、Bilibili 等
- 📄 文章: 博客、技术文档等
- 📜 论文: arXiv、Google Scholar 等
- 📌 其他: 纯文本笔记等

**最佳实践**:
- 看到好资源立即保存，不要拖延
- 添加简短描述，方便日后检索
- 定期清理已完成的资源

---

### 4. 学习统计分析

**适用场景**: 查看学习进度，保持动力。

**使用步骤**:

1. **查看概览**
   - 打开「📊 统计」页面
   - 查看总任务数、已完成数、完成率
   - 选择时间范围（本周/本月/最近 30 天）

2. **难度分布分析**
   - 查看任务难度分布饼图
   - 了解你的学习负载
   - 确保难度均衡

3. **完成率详细**
   - 各难度任务的完成率
   - 简单、中等、困难任务分别统计
   - 找出薄弱环节

4. **学习计划进度**
   - 查看每个计划的完成情况
   - 进度条可视化
   - 及时调整学习策略

**最佳实践**:
- 每周查看一次统计
- 完成率 70% 以上表示进展良好
- 困难任务完成率低是正常的，不要气馁

---

### 5. 快速添加单个任务

**适用场景**: 临时想到一个学习任务，快速添加。

**使用步骤**:

1. 点击「➕ 添加任务」
2. 输入任务名称（如"阅读 React 官方文档"）
3. 选择日期、预计时长、难度、优先级
4. 添加标签（可选）
5. 点击「✅ 添加」

**与批量生成的区别**:
- 批量生成：AI 自动规划整个学习计划
- 快速添加：手动添加单个任务
- 两者可以混合使用

## 🎯 学习模式选择指南

### 交叉学习模式 🔄

**特点**:
- 不同主题交叉安排
- 避免连续学习同一内容导致疲劳
- 适合长期（2-4 周）学习

**适合场景**:
- 学习多个不相关的主题
- 时间跨度较长
- 希望保持学习新鲜感

**示例**:
```
Day 1: Python 基础
Day 2: 数据可视化
Day 3: 机器学习理论
Day 4: Python 进阶
Day 5: Scikit-learn 实践
...
```

---

### 集中攻坚模式 🎯

**特点**:
- 同一主题的任务集中安排
- 适合需要连续深入的学习
- 适合短期（1-2 周）冲刺

**适合场景**:
- 准备考试或面试
- 学习某个特定技术栈
- 需要快速掌握某项技能

**示例**:
```
Day 1-3: Python 基础（连续 3 天）
Day 4-5: 数据处理（连续 2 天）
Day 6-7: 可视化（连续 2 天）
...
```

## 💡 高级技巧

### 1. 利用备份功能

MingDeng 会在以下情况自动备份：
- 批量生成计划
- 批量删除/修改任务
- 手动触发备份

**恢复备份**:
- 进入「⚙️ 配置」→「数据管理」
- 点击「查看备份」
- 选择要恢复的备份

### 2. 调整计划时间

如果计划安排得太紧或太松：
- 打开「首页」→「所有计划」
- 选择要调整的计划
- 批量延后或提前任务

### 3. 导出数据（即将推出）

未来可以导出：
- 学习计划（Markdown 格式）
- 统计报告（PDF 格式）
- 所有数据（JSON 格式）

## 🔧 常见问题解决

### Q1: AI 生成的计划不满意？

**解决方案**:
1. 点击「重新生成」，AI 会生成新的计划
2. 或者保存后手动调整任务
3. 尝试提供更详细的学习内容描述

### Q2: 任务太多，每天完成不了？

**解决方案**:
1. 使用「跳过」功能，不要强迫自己
2. 批量调整计划时间，延长学习周期
3. 删除部分不重要的任务

### Q3: 忘记每天打开 MingDeng？

**解决方案**:
1. 设置浏览器书签，方便打开
2. 每天早上第一件事：打开 MingDeng
3. 未来会支持桌面应用和通知功能

### Q4: 资源没有自动关联到任务？

**解决方案**:
1. 检查 API 配置是否正确
2. 手动关联资源到任务
3. 在保存资源时添加更详细的描述

## 📖 学习建议

### 1. 制定合理的学习计划

- 不要贪多，从小目标开始
- 每天 1-2 小时的学习比 8 小时冲刺更有效
- 留出缓冲时间，不要排满

### 2. 保持学习节奏

- 每天固定时间学习
- 完成任务后及时标记
- 定期查看统计，保持动力

### 3. 灵活调整

- 计划赶不上变化，及时调整
- 不要因为跳过任务而有负罪感
- MingDeng 是辅助工具，不是监工
